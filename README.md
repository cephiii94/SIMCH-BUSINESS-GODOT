# 🎮 SimCH Business

A long-term business simulation game built with **Godot 4** using **GDScript**.

## 🚀 Project Overview

SimCH Business is designed with a highly modular, decoupled architecture, allowing every system (Time, Economy, Inventory, Warehouse, etc.) to be easily reused in future SimCH simulation games.

## 🛠️ Tech Stack & Requirements

- **Game Engine**: Godot Engine 4.3 (or compatible 4.x versions)
- **Language**: GDScript (with strict static typing)
- **Target Platforms**: PC / Mobile (Android)

## 📁 Directory Structure

- `src/` - Game source code and scene files.
  - `src/autoload/` - Global singleton managers (e.g., EventBus, TimeManager).
  - `src/core/` - Data structures, base resources, core logic.
  - `src/ui/` - Interface menus, HUD components, themes.
  - `src/world/` - Game world, levels, camera control.
- `assets/` - Static files (textures, audio, fonts).
- `docs/` - Documentation, design documents, and coding guidelines.
